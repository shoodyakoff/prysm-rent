# PRYSM - Макет "Метрики бизнес-центра"

Этот проект представляет собой Pixel Perfect верстку главной страницы модуля "Аренда" для системы PRYSM.
Проект реализован на чистом HTML, CSS и JS без использования фреймворков, как указано в ТЗ.

## Структура проекта

- `index.html` - Главная страница.
- `css/style.css` - Все стили (цвета, сетки, адаптивность).
- `js/main.js` - Логика интерактивности (клики, иконки).
- `js/data.js` - (Зарезервировано) Файл для будущих данных договоров.

## Как запустить

Просто откройте файл `index.html` в любом современном браузере (Chrome, Safari, Firefox).
Установка зависимостей не требуется. Иконки подгружаются через CDN (Lucide Icons).

## Функционал

1. **Адаптивная верстка**:
   - Фиксированная шапка и сайдбар.
   - Резиновая сетка метрик и отчетов.
2. **Интерактивность**:
   - Hover-эффекты на карточках и меню.
   - Клик по метрикам выводит название метрики в консоль браузера.
   - Клик по активным отчетам выводит название отчета в консоль.
   - Переключение активного пункта меню (визуально).
3. **Дизайн**:
   - Соответствие цветовой палитре и размерам из ТЗ.
   - Градиентная буква "Y" в логотипе.
   - Стилизованные скроллбары (нативные браузерные).

## Примечания

- Используются иконки [Lucide](https://lucide.dev/).
- Шрифты подгружаются с Google Fonts (Inter).
